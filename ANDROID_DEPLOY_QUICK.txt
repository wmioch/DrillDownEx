================================================================================
                              FIRST TIME SETUP
================================================================================

If this is your first deployment run:

1. Phone Setup:
   - Settings > About Phone > Build Number (tap 7 times)
   - Settings > Developer options > Enable USB Debugging
   - Allow the security prompt on phone

2. Computer Setup:
   - Android SDK/NDK installed
   - ANDROID_HOME environment variable set
   - Keystore file created and gradle.properties updated

3. First Deploy:
   - Connect phone via USB
   - Double-click: DEPLOY_ANDROID.bat
   - Wait for build and install to complete

================================================================================
                           IF SOMETHING GOES WRONG
================================================================================

ERROR: "No Android device connected"
   Fix: 
   - Check USB cable connection
   - Enable USB Debugging on phone (Settings > Developer options)
   - Try different USB port
   - Restart phone

ERROR: "Build failed"
   Fix:
   - Run: gradlew --stop
   - Run DEPLOY_ANDROID.bat again

ERROR: "Installation failed"
   Fix:
   - Phone storage full? Clear some space
   - Run: adb uninstall de.dakror.quarry
   - Run DEPLOY_ANDROID.bat again

ERROR: "ADB not available"
   Fix:
   - Set ANDROID_HOME environment variable
   - Add %ANDROID_HOME%\platform-tools to PATH
   - Restart terminal/computer

================================================================================
                            USEFUL COMMANDS
================================================================================

Check connected devices:
   adb devices

View phone logs:
   adb logcat | findstr "quarry"

Uninstall app from phone:
   adb uninstall de.dakror.quarry

Just build (without deploy):
   gradlew android:assembleFullRelease

Manual install:
   adb install -r android\build\outputs\apk\full\release\android-full-release.apk

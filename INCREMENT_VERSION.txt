================================================================================
              INCREMENT VERSION BEFORE EACH ANDROID DEPLOYMENT
================================================================================

BEFORE you run DEPLOY_ANDROID.bat, you MUST increment the version number!

Otherwise, Android won't recognize it as an update.

================================================================================
                              QUICK STEPS
================================================================================

1. Open: D:\Projects\DrillDown\android\build.gradle

2. Find lines 46-47:
   versionCode 122
   versionName "v122"

3. Change to (increment by 1):
   versionCode 123
   versionName "v123"

4. Save the file (Ctrl+S)

5. Double-click: DEPLOY_ANDROID.bat

================================================================================
                            DETAILED EXAMPLE
================================================================================

BEFORE (Current):
---
    defaultConfig {
        applicationId "de.dakror.quarry"
        minSdkVersion 19
        targetSdkVersion 30
        versionCode 122              ‚Üê UPDATE THIS
        versionName "v122"           ‚Üê UPDATE THIS
        multiDexEnabled false
    }
---

AFTER (Updated for deployment):
---
    defaultConfig {
        applicationId "de.dakror.quarry"
        minSdkVersion 19
        targetSdkVersion 30
        versionCode 123              ‚Üê INCREMENTED
        versionName "v123"           ‚Üê INCREMENTED
        multiDexEnabled false

---

Then deploy with DEPLOY_ANDROID.bat

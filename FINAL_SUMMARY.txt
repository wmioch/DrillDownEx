================================================================================
                           TROUBLESHOOTING
================================================================================

GAME WON'T START:
   → Make sure you're running RUN_GAME.bat (it sets the working directory)
   → Alternative: cd D:\Projects\DrillDown\android\assets && java -jar ..\desktop\build\libs\desktop-1.0.jar

BUILD FAILS:
   → Run: gradlew.bat --stop
   → Delete: .gradle\ folder
   → Retry: BUILD_GAME.bat

OUT OF MEMORY:
   → Edit gradle.properties
   → Change: org.gradle.jvmargs = -Xms512m -Xmx4096m

ASSETS NOT FOUND:
   → Verify android\assets\ directory exists with game resources
   → Don't move the project folder
